### Tweets 3 through [N-1]: THE MEAT

Repeat this block for each item:

```
[Number]/[Total]: **[Bold title/name]**

[1-2 sentences: what it does + why it matters]

[If prompt-style content:]
"[The actual prompt with [YOUR X] placeholders]"

[If tool recommendation:]
[Tool name], [one-line description], [free/paid]

[Optional: screenshot attachment]
```

**Rules for the meat:**
- One idea per tweet, one tweet per idea
- Bold the key phrase in every tweet
- Use line breaks: no dense paragraphs
- Include `[YOUR X]` placeholders in all prompts
- Every tweet should be screenshot-worthy on its own

## Post-Publish Checklist

- [ ] Reply to every comment in the first 30 minutes
- [ ] Quote-tweet the thread with a different hook angle (1-2 hours later)
- [ ] Pin the thread to your profile if it's performing
- [ ] Add a "bonus" reply tweet if engagement is building (hour 3-6)
- [ ] Repost or remix the best-performing tweet the next day
- [ ] Track metrics: impressions, bookmarks, bookmark/like ratio, replies

## Quick Variants

### Resource List Thread (tool roundups, prompt packs)
- Hook: Shock number + value promise
- Body: Numbered items, one per tweet, categorized
- Close: Multi-CTA + comment-DM bait
- Length: 8-15 tweets

### Tutorial Thread (how-to, workflow)
- Hook: Transformation claim or anti-overwhelm
- Body: Step-by-step numbered process
- Close: Recap + "start with step 1 today"
- Length: 5-10 tweets

### Analysis Thread (data, contrarian take)
- Hook: Curiosity gap or contrarian
- Body: Evidence, examples, framework
- Close: Summary + provocative question for replies
- Length: 5-8 tweets

### Story Thread (personal experience, case study)
- Hook: Transformation or pain recognition
- Body: Chronological narrative with lessons
- Close: Key takeaway + CTA
- Length: 5-12 tweets
