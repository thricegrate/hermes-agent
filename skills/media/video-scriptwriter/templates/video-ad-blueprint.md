## Pre-Shoot Checklist

### Script
- [ ] Hook scripted and rehearsed (read aloud 3+ times)
- [ ] Full script reviewed for word count vs duration
- [ ] Body copy written and reviewed separately
- [ ] All text overlay copy finalized and spell-checked
- [ ] A/B hook variation scripted

### Talent
- [ ] Expert wardrobe selected (solid color, no logos)
- [ ] Testimonial contributors briefed on what to say
- [ ] Energy direction reviewed per act

### Production
- [ ] Background / setting prepared for each act
- [ ] Lighting tested (especially for expert segments)
- [ ] Audio tested (lapel mic charged, backup available)
- [ ] Camera angles planned (A-angle and B-angle marked)
- [ ] Visual aids / diagrams designed and ready
- [ ] Screen recordings captured (if Demo Proof framework)

### Post-Production
- [ ] Editing rhythm planned per act
- [ ] Text overlay styles defined (font, color, position)
- [ ] Music track selected
- [ ] Transition sounds prepared
- [ ] Platform-specific safe zones noted for text placement
