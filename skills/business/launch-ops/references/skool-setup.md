# Skool Setup Reference

## Why Skool

- Simple, fast, intuitive
- Intentional constraints prevent overthinking unimportant details
- Focus on creating content and ensuring consumption, not button colors
- Flat $99/month - all features included, unlimited everything, no per-customer fees
- Built-in community feed, calendar, and gamification

## Skool Setup Checklist (Detailed)

### 1. Create Skool Group
- Sign up at Skool
- One plan: $99/mo, unlimited everything

### 2. Group Branding
- **Settings > General**
- Upload group Icon (square logo, 256x256px recommended)
- Upload group Cover (banner, 1920x480px recommended)
- Write Group Description (what the product is, who it's for, what they'll learn)

### 3. Discussion Categories (Optional)
- **Settings > Categories**
- Suggested categories:
  - Introductions
  - Wins & Celebrations
  - Questions & Help
  - Resources & Tools
  - General Discussion

### 4. Classroom Modules
- Navigate to **Classroom** section
- Create a "course" for each major module
- Arrange in logical progression (drag to reorder)
- Add banner image to each module (use Canva - 1280x720px)

### 5. Upload Content
- Add lessons to each module
- Clear, descriptive titles
- Upload videos, PDFs, or other resources
- Include descriptions for each lesson
- Set drip schedule if desired (optional)

### 6. Start Here Module
Create as the FIRST module with:
- Welcome message
- How to navigate the product
- Community guidelines
- Quick-start checklist
- Links to important resources

### 7. Pinned Post
Create and pin a post in the community feed with:
- Welcome and orientation
- Link to Start Here module
- Important links and resources
- How to get help

### 8. Automated Welcome DM
- **Settings > Plugins > Auto DM new members**
- Write welcome DM with:
  - Personal greeting
  - Link to pinned post
  - Encouragement to introduce themselves
- Toggle ON

## Budget Alternative: Notion

### When to Use
- Just starting out, tight budget
- Don't want $99/month commitment yet
- Want to launch fast and iterate

### Setup
1. Create new Notion workspace or page
2. Build module structure using nested pages
3. Add lesson content to each page
4. Create Start Here landing page
5. Set sharing to "Anyone with the link can view"

### Limitation
- No built-in access control (someone could share the link)
- This is fine when starting out - focus on shipping, not perfecting
