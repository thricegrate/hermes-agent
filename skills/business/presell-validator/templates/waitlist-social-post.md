# Waitlist Social Post Templates

Compressed version of the announcement email, formatted for each platform. Same goal: get people to signal interest.

---

## X / Twitter Version

```
[HOOK -- curiosity or bold claim, under 15 words]

I'm building [PRODUCT NAME]: [one-line description].

It's for [WHO] who want to [OUTCOME].

What's inside:
- [Fascination 1]
- [Fascination 2]
- [Fascination 3]

Not ready yet, but you can grab early access:

[WAITLIST LINK]
```

**Alt CTA (comment-gated):**
```
Drop "EARLY" in the replies and I'll send you the link when it's live.
```

## Instagram Caption Version

```
[HOOK -- first line must stop the scroll]

I've been working on something behind the scenes.

[PRODUCT NAME] -- [one-line description].

If you [PAIN POINT], this is built for you.

Link in bio to join the waitlist for early access.

(Or DM me "[KEYWORD]" and I'll send it to you directly)
```

---

## Template Rules

- Hook = first line. If it doesn't stop the scroll, nothing else matters.
- Short sentences. Lots of white space. No walls of text.
- One CTA per post. Don't split attention.
- Comment-gated CTAs ("write X in comments") boost engagement AND qualify interest. Use when you want both reach and signal.
- Don't oversell something that doesn't exist yet. "I'm building" > "I'm launching."
- Adapt tone to the platform, but keep it conversational everywhere.
- Post across all active platforms within the same 24 hours as the email blast.
- Cross-ref: use `content-social` skill for platform-specific formatting rules.
