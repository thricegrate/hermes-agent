# Production Checklist: [Video Title]

## Pre-Production

### Concept Lock
- [ ] Title and thumbnail drafted and reviewed
- [ ] Format selected and retention structure planned
- [ ] Wow factor identified
- [ ] "Excite Me" test passed (creator is genuinely excited)
- [ ] Simplicity test passed (teenager could follow every beat)

### Critical Components
- [ ] All critical components identified and listed
- [ ] Each critical component has a named owner
- [ ] Each critical component has a backup plan
- [ ] Critical components are being checked daily (minimum)
- [ ] Speak up the second anything goes wrong (do not wait)

### Feasibility
- [ ] Feasibility study completed for all critical components
- [ ] Confirmed: the key elements of this video are actually possible
- [ ] If feasibility fails, video stays in concept phase (no one else wastes time)

### Logistics
- [ ] Location(s) secured with backup day option
- [ ] Equipment/props sourced and verified (request videos from vendors)
- [ ] Contractors/vendors checked in on daily (not weekly, not at deadline)
- [ ] Consultants contacted for any unfamiliar elements
- [ ] Budget approved and tracked

### Creative
- [ ] Minute-by-minute retention script completed
- [ ] Re-engagement beats planned (3-min and 6-min marks)
- [ ] Brand deal integration planned (if applicable)
- [ ] Opening dialogue scripted word-for-word
- [ ] All elements match the title/thumbnail expectation contract

### Bottleneck Check
- [ ] Bottleneck person/step identified
- [ ] They know they are the bottleneck (told directly)
- [ ] Deadline set and agreed upon
- [ ] Daily check-ins scheduled (do not wait for the deadline)

---

## Production Day

### Before Cameras Roll
- [ ] Critical components re-verified on-site
- [ ] Lighting checked (especially for the first minute; best lighting here)
- [ ] Audio checked
- [ ] All props/elements match what was planned
- [ ] Title/thumbnail elements present and accurate
- [ ] Team briefed on the retention script and key moments

### During Filming
- [ ] Video everything: all key moments, scouting, progress, problems
- [ ] Follow the retention script: know what minute mark you are filming for
- [ ] If something goes wrong, check if it can become content
- [ ] Do not leave contestants/participants waiting more than 3 hours
- [ ] Maintain proper communication lines (go up, then over)
- [ ] Keep the set minimal (if someone is not necessary, they should not be there)

### Backup
- [ ] Backup day available if needed
- [ ] Backup footage captured for key moments
- [ ] If extending, all vendors/locations confirmed for extension

---

## Post-Production

### Edit
- [ ] Edit follows the retention structure (first minute is tightest)
- [ ] "No dull moments" pass: scrub every second, cut anything boring
- [ ] Re-engagement beats land at the planned minute marks
- [ ] Brand deal segment integrated naturally (no retention crater)
- [ ] Ending is abrupt: cut immediately after the payoff
- [ ] No "wrapping up" or goodbye at the end

### Review
- [ ] Watch the full edit start to finish without pausing
- [ ] Note every moment where your attention wanders; fix those
- [ ] Does the first minute match the title/thumbnail promise?
- [ ] Is the wow factor visible and impactful?
- [ ] Would you watch this all the way through if it was not your video?

### Thumbnail & Title Finalization
- [ ] Final thumbnail created and reviewed at mobile size
- [ ] Final title selected from options
- [ ] Thumbnail + title tested with 3+ people for gut reactions
- [ ] "Say the negatives" critique completed (why would someone NOT click?)

---

## Launch

### Upload
- [ ] Upload scheduled or published
- [ ] Description includes email opt-in / newsletter link
- [ ] Tags and metadata completed
- [ ] End screen elements added
- [ ] Community post published (teaser or discussion prompt)

### Promotion
- [ ] 3-5 short-form clips cut and scheduled (TikTok, Reels, Shorts)
- [ ] Social media posts scheduled (teaser, behind-the-scenes)
- [ ] Newsletter tie-in prepared (if applicable)

### Performance Tracking
- [ ] Monitor 1-out-of-10 rating in first few hours
- [ ] Check retention curve for drop-off points
- [ ] Note what worked and what did not for future videos
- [ ] Log learnings: format, title/thumbnail performance, audience response
