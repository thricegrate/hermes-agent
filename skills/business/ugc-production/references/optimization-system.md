# Weekly Optimization System

The intelligence layer that converts daily performance data into actionable creative decisions for every subsequent production cycle.

---

## The Monday Review Process

### Step 1: Identify Top 5 and Bottom 5

Rank all content from the previous week by click-through rate (CTR). For each of the top 5 and bottom 5, document:

| Variable | Top Performer | Bottom Performer |
|----------|--------------|-----------------|
| Hook category | [1-5] | [1-5] |
| Avatar type | [A-D] | [A-D] |
| Script concept | [name] | [name] |
| Length | [15/30/45s] | [15/30/45s] |
| Platform | [IG/TT/YT] | [IG/TT/YT] |
| Account | [name] | [name] |
| Watch-through rate | [%] | [%] |
| Save rate | [%] | [%] |
| CTR | [%] | [%] |
| Estimated revenue | [$] | [$] |

### Step 2: Extract the Pattern

**Ask these questions:**
- What hook categories appear most in the top 5? Least in the bottom 5?
- Which avatar types correlate with high performance?
- Is there a length that consistently over/underperforms?
- Are specific accounts outperforming others with the same content?
- Is a specific platform delivering better results?

**Write the pattern insight in one sentence.** Example: "Counterintuitive hooks with relatable avatars at 30s length consistently outperform other combinations. Before-state immersion hooks underperform on TikTok but work well on Instagram."

This pattern insight is the single most valuable output of the weekly review.

### Step 3: Update the Variation Testing Matrix

**Sufficiently tested (draw conclusions):**
- Variables with 10+ data points per combination
- Clear winner with >20% performance gap
- Action: double down on winners, retire losers

**Insufficient data (need more testing):**
- Variables with <10 data points
- No clear winner (within 10% of each other)
- Action: increase test volume for this combination next week

**New combinations suggested by data:**
- Combinations the pattern insight points to but haven't been tested yet
- Action: add to next week's production schedule

### Step 4: Set Next Week's Production Priorities

Based on the pattern insight:

1. **Double down:** Allocate 60% of production to winning combinations
2. **Continue testing:** Allocate 25% to combinations with insufficient data
3. **Explore:** Allocate 15% to new combinations suggested by the pattern

## The North Star Metric

**Revenue Per Piece of Content Posted (RPCP)**

```
RPCP = Total Weekly Revenue / Total Pieces Published
```

This single metric captures the combined effectiveness of every element: content quality, hook performance, avatar selection, funnel conversion, email recovery, and offer strength.

| Trend | Meaning | Action |
|-------|---------|--------|
| Increasing 3+ weeks | System is learning and improving | Maintain current approach, begin considering scale |
| Flat 2+ weeks | Optimization has stalled | Deep review of quiz conversion, email sequences, and hook quality |
| Declining 2+ weeks | Something broke | Diagnose layer by layer (content -> funnel -> email) |

---

## Layer-by-Layer Diagnosis

When RPCP is flat or declining, diagnose which layer is underperforming:

### Layer 1: Content (Views + Watch-Through)
**Metrics:** Views per piece, watch-through rate
**If declining:** Hook quality degrading, audience fatigue on current concepts, account health issues
**Fix:** Rotate in new hook categories, test new concepts, review account health

### Layer 2: Traffic (Clicks + CTR)
**Metrics:** Link clicks, click-through rate
**If declining:** CTA not compelling, link-in-bio friction, quiz entry not clear
**Fix:** Test new CTA formats, simplify link-in-bio, test DM automation entry point

### Layer 3: Conversion (Quiz + Purchase)
**Metrics:** Quiz start rate, completion rate, recommendation-to-purchase rate
**If declining:** Quiz fatigue, recommendation screen weak, offer misaligned
**Fix:** A/B test quiz questions, rewrite recommendation screens, review offer economics

### Layer 4: Recovery (Email + DM)
**Metrics:** Email sequence conversion, DM follow-up response
**If declining:** Email deliverability, subject line fatigue, sequence not addressing objections
**Fix:** Refresh email subject lines, update objection handling, check deliverability

## Monthly Performance Review (First Monday of Each Month)

In addition to the weekly review, conduct a monthly deep review:

1. **Revenue trend:** Is monthly revenue growing, flat, or declining?
2. **Scale readiness:** Do all 3 scale criteria pass? (4 weeks consistent, 5%+ quiz CVR on 500+, 3+ validated combinations)
3. **Portfolio health:** Tier distribution correct? Warmth pipeline full? Any chronic underperformers?
4. **Content efficiency:** Is RPCP improving month-over-month?
5. **Audience intelligence:** What new insights has quiz data revealed about the audience?
6. **Creative refresh:** Are any hook categories or concepts showing fatigue (declining performance despite consistent posting)?

## Optimization Spreadsheet Structure

### Tab 1: Portfolio Overview
Account name, tier, creation date, warmth status, daily volume, follower count, 7-day avg watch-through, save rate, CTR, elevation target, notes

### Tab 2: Content Performance
Content ID, concept, hook type, avatar ID, length, register, platform, account, post date, 48h watch-through, 7d watch-through, 7d save rate, 7d CTR, 7d estimated revenue, notes

### Tab 3: Weekly Health
Week, total posts, total impressions, avg watch-through, total clicks, quiz completions, purchases, revenue, RPCP, notes

### Tab 4: Variation Test Results
Test ID, variable tested, variation A description, variation B description, variation A CTR, variation B CTR, winner, insight extracted, applied to next cycle?
