# Newsletter Ad Prompt Library

12 ready-to-use prompt templates for Nano Banana Pro (Gemini). Two variants per layout template. Fill placeholders before use.

## Placeholders

| Placeholder | Description | Example |
|---|---|---|
| `{HEADLINE}` | Main headline text | "Unlock 200+ AI-Powered Income Ideas" |
| `{SUBTITLE}` | Supporting line | "Free guide: Turn AI tools into real revenue streams" |
| `{PRODUCT_TYPE}` | Mockup type | ebook, laptop, document pages, clipboard |
| `{PRIMARY_COLOR}` | Main background/text color with name | "navy (#1B2838)" |
| `{ACCENT_COLOR}` | Accent color with name | "orange (#FF6B35)" |
| `{ASPECT_RATIO}` | Image dimensions | "16:9 landscape" |

---

## Template 1A: Clean Ebook (Standard)

```
Generate a {ASPECT_RATIO} image.

Background: solid cream (#FFF5EB), clean and uniform.

LEFT SIDE (55% of image):
- Bold serif headline text in {PRIMARY_COLOR} that says EXACTLY: "{HEADLINE}"
- Below it, smaller sans-serif subtitle text in gray (#666666) that says EXACTLY: "{SUBTITLE}"
- Text is left-aligned with generous padding from the left edge.

RIGHT SIDE (45% of image):
- A 3D {PRODUCT_TYPE} mockup, tilted 10-15 degrees to the right
- The {PRODUCT_TYPE} has a simple, clean cover design with abstract geometric shapes
- Behind the {PRODUCT_TYPE}: a large solid {ACCENT_COLOR} circle, partially cropped by the right edge
- The {PRODUCT_TYPE} casts a subtle drop shadow

Style: clean, professional, modern editorial. Crisp edges, no texture on background.

NO logos. NO brand names. NO company names. NO watermarks. NO borders. NO frames. NO additional text beyond the headline and subtitle.
```
