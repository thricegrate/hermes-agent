## One-Time Catalyst Tripwires

These are events that don't have a scheduled cadence but would materially change the thesis if they fire.

### CONFIRMING events (reinforce thesis, upgrade conviction)

- **New $100B+ cluster announcement** (any source) → reinforces Ch.3 trajectory
- **New hyperscaler-utility long-term PPA** (Microsoft, Amazon, Google, Meta with any US power producer) → reinforces nuclear/power thesis
- **AGI announcement from a major lab** (rigorous, not marketing) → all Tier 1 to max conviction
- **Executive order establishing government AGI project** → Ch.7 confirmation, PLTR/BAH/LDOS upgrade
- **NRC permits multiple SMRs** → URA/OKLO/NNE upgrade
- **Major export-control tightening** (additional China bans) → semis moat widens
- **Confirmed Chinese AI lab weights theft** → Ch.4 confirmation, HACK/PLTR/defense IT upgrade
- **Hyperscaler capex guides UP >25% from prior quarter** → SMH conviction max

### THREATENING events (potential kill switches — see [sizing-overlay.md](sizing-overlay.md))

- **Major model release shows capability plateau** → reduce semi/AI tier sizing
- **Hyperscaler capex guide cut >15%** → exit Tier 3, reduce Tier 2
- **NVDA datacenter rev YoY <30% for 2 quarters** → drop Tier 1 to Tier 2
- **MU HBM rev guidance down 2+ quarters** → exit HBM names
- **Regulatory pause / training cap order** → cut all sizing to base
- **Visible alignment incident** (deceptive model behavior in deployment) → reduce all + watch HACK
- **Taiwan Strait conflict escalation** → exit TSM + ASML, watch defense + cyber for upside
- **DeepSeek-class efficiency breakthrough at frontier scale** → semis demand reprices down
- **CXMT (China memory) production breakthrough** → MU/SNDK margin compression
