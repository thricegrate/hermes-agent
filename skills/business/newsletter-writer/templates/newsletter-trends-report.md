# Newsletter Template: Trends Report (Chasing Creative Style)

Use this template for time-sensitive industry shifts and emerging patterns. NOT evergreen (unlike deep dives). Best for: "here's what changed this week and why you should care."

**Distinct from Roundup:** Trends Report interprets shifts (~150 words per section, image-led). Roundup covers stories (TLDR + bullets + editor's take, story micro-template).
**Distinct from Deep Dive:** Trends Report covers 3-5 shifts briefly. Deep Dive covers one topic in depth.

## Writing Notes

**Target length:** 500-800 words total

**Tone:** Cheeky but clear. Think smart friend who reads everything and tells you what actually matters. Blake Wisz's phrase for this: "real not clever." If it sounds like you're trying to be witty, cut it. If it sounds like you're being honest, keep it.

**Section length:** ~150-160 words per trend. Tight. If you need more than 200 words for a trend, it's a deep dive, not a trends report.

**Images:** EVERY trend section starts with a visual. This format is image-led. No image = no trend section.

**NOT evergreen:** This content is timely. Don't try to make it last forever. Its value is freshness. A trends report from last month should feel dated. That's correct.

**Frequency:** Weekly or bi-weekly. Pairs well with a roundup schedule (roundups for daily coverage, trends report for weekly analysis).

**Resource at bottom:** This is where you naturally link to your own tools, lead magnets, or products. Frame it as genuinely helpful for acting on the trends, not as a hard sell.

**Trend count:** 3 is the minimum. 4-5 for bigger weeks. Never more than 5 (that's a roundup). Never fewer than 3 (that's a deep dive).

**Swipe this section format:**
- [Headline]
- Quick context (1-2 lines)
- Why it matters (1-2 lines)
- Optional: Link to learn more or tool/resource

**When to use Trends Report vs other templates:**
- Trends Report when highlighting shifts and patterns (WHY things matter)
- Roundup when covering multiple stories (WHAT happened)
- Deep Dive when one topic needs in-depth analysis
- Curation when collecting best links with commentary
