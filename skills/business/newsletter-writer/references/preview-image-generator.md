# Newsletter Preview Image Prompt Generator

Generate 10 ChatGPT/DALL-E image prompts for newsletter header or preview images. Each prompt produces an award-winning, non-ordinary visual that captures the newsletter's central idea.

---

## Rules

- 10 varied image concepts per newsletter issue
- Always set aspect ratio to 16:9
- Add other relevant tags where needed (style, lighting, mood, etc.)
- Award-winning quality, non-ordinary compositions
- Simple and straightforward concepts
- Challenge yourself to find unexpected visual angles
- Check all concepts against `references/banned-words.md` banned visual concepts section

## Banned Visual Concepts

Never use these overused concepts (full list in `references/banned-words.md`):
cathedral, storm, commute, roof-top, prism, conveyor, origami, maze, control room, split screen, sticky notes

## Process

1. Analyze the newsletter text
2. Find the central idea
3. Generate 10 completely different visual interpretations
4. Format each as a ready-to-paste ChatGPT image prompt with ar 16:9 and relevant tags

## When to Use

After the newsletter body and subject line are finalized, run this generator to create header image options. Pick the one that best matches the issue's emotional tone and topic.

---

## Ready-to-Use Prompt Template

Paste the newsletter text after the prompt and run:

```
Think differently. Now analyse the text, find the central idea, and give me 10 ChatGPT create image prompts for this idea. Award-winning, non-ordinary, simple and straighforward. Challenge yourself. No "cathedral", "storm", "commute", "roof-top", "prism", "conveer", "origami", "maze", "control room", "split screen", "sticky notes" etc. Always set ar to 16:9, add other tags where relevant.
```
