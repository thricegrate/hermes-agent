# Exercise Type Matrix

## Overview

Exercise type is determined by two factors:
1. **Learner stage** (which day in the 28-day challenge)
2. **AI tool category** (text, image, or audio)

The intersection determines the exact exercise pattern.

---

## Exercise Types by Stage

### Type 1: Multiple Choice (Days 1-7, Zero Confidence)

**Cognitive load:** Minimal. The learner recognizes, not generates.

**Design pattern:**
- Present 2-4 options (2 is ideal for speed, 4 max for difficulty)
- One option demonstrates the concept, others don't
- Distractors should be plausible (common mistakes, not obviously wrong)
- Always explain why the correct answer is right AND why the top distractor is wrong

**Subtypes:**
| Subtype | Description | When to Use |
|---------|-------------|-------------|
| **Output comparison** | "Which output was generated by the better prompt?" | Text tools (ChatGPT, Gemini) |
| **Screenshot match** | "Which image was generated by this prompt?" | Image tools (DALL-E, Midjourney) |
| **Scenario match** | "Which tool is best for this task?" | Audio tools, cross-tool lessons |
| **Spot the technique** | "Which prompt uses [concept name]?" | Any tool, concept reinforcement |

**Example — Output comparison (ChatGPT, Day 2):**
```
Which email was written using a role prompt?

A) "I am writing to inform you that the project deadline
    needs to be extended due to various circumstances..."

B) "Quick flag on the Henderson deliverable — we hit a
    dependency blocker Tuesday that pushed critical path
    out by 3 days. Requesting we move to 3/19..."

Answer: B
Why: Specific details (project name, days, new date) and
professional shorthand indicate ChatGPT was given a
project manager role. A is generic because no role was set.
```

**Example — Screenshot match (DALL-E, Day 5):**
```
This prompt was used: "A coffee shop interior, warm
watercolor illustration style, soft lighting"

Which image matches?
A) [Photorealistic coffee shop photo]
B) [Watercolor-style illustration with warm tones]
C) [Cartoon/clipart coffee shop]

Answer: B
Why: "Watercolor illustration style" is a style anchor.
A lacks the illustration quality. C is cartoon, not watercolor.
```

### Type 2: Fill-in-the-Blank (Days 8-14, Building Confidence)

**Cognitive load:** Low-medium. The learner generates one element within a structure.

**Design pattern:**
- Provide a complete prompt template with exactly 1 blank
- The blank should target the specific concept being taught
- Give context about the desired outcome so the blank isn't ambiguous
- Accept multiple valid answers (provide 2-3 strong examples in the reveal)

**Subtypes:**
| Subtype | Description | Blank Position |
|---------|-------------|---------------|
| **Role blank** | "Act as a ___ and..." | Beginning of prompt |
| **Context blank** | "...considering that ___..." | Middle of prompt |
| **Format blank** | "...Format the output as ___" | End of prompt |
| **Modifier blank** | "[subject], ___ style" | Style/quality position |
| **Setting blank** | "Enable ___ before recording" | Tool configuration |

**Example — Role blank (ChatGPT, Day 9):**
```
You want ChatGPT to help you write a cover letter for a
marketing manager position.

Complete this prompt:
"Act as a ___ . Review my resume [attached] and write a
cover letter for the Marketing Manager role at Spotify.
Highlight my campaign experience."

Strong answers:
- "hiring manager at a tech company"
- "senior recruiter specializing in marketing roles"
- "career coach with 15 years in tech recruiting"

Weak answers:
- "helpful assistant" (too generic, no domain expertise)
- "writer" (too broad, no hiring context)
```

**Example — Modifier blank (DALL-E, Day 11):**
```
You want a hero image for a meditation app. The brand is
calm, minimal, and premium.

Complete this prompt:
"A person meditating on a mountain at sunrise, ___ ,
soft pastel color palette, wide aspect ratio"

Strong answers:
- "editorial photography style, shallow depth of field"
- "minimal 3D render, soft volumetric lighting"
- "fine art photography, Hasselblad medium format look"

Weak answers:
- "beautiful" (subjective, not a style anchor)
- "high quality" (every prompt should be high quality)
```

### Type 3: Prompt Rewrite (Days 15-21, Practicing)

**Cognitive load:** Medium-high. The learner diagnoses a problem and fixes it.

**Design pattern:**
- Provide a complete but flawed prompt
- The flaw must be specific to today's concept (not generally bad)
- State the desired outcome so the learner knows what "fixed" looks like
- The fix should require changing/adding 5-15 words, not a total rewrite

**Subtypes:**
| Subtype | Description | Flaw Type |
|---------|-------------|-----------|
| **Missing technique** | Prompt lacks today's concept entirely | Addition needed |
| **Wrong technique** | Prompt uses the concept incorrectly | Replacement needed |
| **Vague technique** | Prompt uses a weak version of the concept | Strengthening needed |
| **Conflicting instructions** | Prompt has contradictory elements | Resolution needed |

**Example — Missing technique (ChatGPT, Day 16):**
```
This prompt produces generic, unfocused output.
Rewrite it using The Constraint Prompt (today's concept):

Original: "Write me 5 social media post ideas for
my bakery"

The output was: generic posts like "Try our fresh bread!"
that could apply to any bakery.

Your goal: Get posts specific to a sourdough bakery in
Portland that just won a local food award.

Ideal rewrite: "Write 5 Instagram post ideas for a
sourdough bakery in Portland that just won the
Willamette Week Best Bakery award. Each post must
mention sourdough, be under 150 characters, and
include one emoji. Tone: proud but humble."

Key fix: Adding specific constraints (location, award,
character limit, tone) forces specific output.
```

**Example — Vague technique (DALL-E, Day 18):**
```
This DALL-E prompt produces inconsistent results.
Strengthen the style anchor:

Original: "A modern office space, nice style, good
lighting, professional look"

Problem: "nice," "good," and "professional" are
subjective — DALL-E interprets them randomly each time.

Ideal rewrite: "A modern office space, architectural
photography style, natural window lighting with soft
shadows, Kinfolk magazine aesthetic, shot on 35mm film"

Key fix: Replace every subjective adjective with a
specific visual reference.
```

### Type 4: Write from Scratch (Days 22-28, Fluent)

**Cognitive load:** High. The learner synthesizes multiple concepts.

**Design pattern:**
- Provide a real-world scenario with a clear goal
- Specify 1-2 constraints (so it's not open-ended chaos)
- The task should require combining today's concept with at least one previous concept
- Provide a rubric (3-4 criteria) rather than a single "correct" answer

**Subtypes:**
| Subtype | Description | Complexity |
|---------|-------------|-----------|
| **Single-tool mastery** | Complete prompt for one tool | 2 concepts combined |
| **Multi-step workflow** | Chain of prompts for one goal | 3+ steps |
| **Cross-tool pipeline** | Use 2+ tools together | Tool handoff required |
| **Real-world project** | End-to-end deliverable | Full synthesis |

**Example — Single-tool mastery (ChatGPT, Day 23):**
```
Scenario: You're a freelance copywriter. A client needs
3 email subject lines for a Black Friday sale on
premium headphones.

Write a complete ChatGPT prompt that uses:
1. The Role Prompt (Day 3)
2. The Constraint Prompt (Day 16)

Rubric:
✓ Assigns a specific, relevant role
✓ Includes at least 3 specific constraints
✓ Specifies the output format
✓ Could be copy-pasted into ChatGPT and work immediately

Example strong answer:
"Act as a direct-response copywriter who has written
email campaigns for Bose and Sony. Write 3 Black Friday
email subject lines for premium noise-canceling
headphones priced at $349 (now $249). Constraints: each
under 50 characters, must include a number, no ALL CAPS,
create urgency without being spammy. Format: numbered
list with a one-line rationale for each."

+50 XP
```

## Exercise Design Rules by Category

### Writing (ChatGPT, Gemini)
- Always provide the writing context (audience, purpose, tone)
- Exercises should produce copy that could be used immediately
- Compare outputs side by side when possible
- Grade on specificity: specific > generic, always

### Images (DALL-E, Midjourney)
- Reference visual outputs even in text-based exercises
- Style anchors should reference real visual traditions, not adjectives
- Test composition vocabulary: framing, lighting, perspective, palette
- For Midjourney-specific: include aspect ratio and parameter flags

### Save Time (Otter.ai, workflow tools)
- Focus on settings and configurations, not just "use the tool"
- Measure time saved: always quantify (e.g., "45 min meeting = 30 sec summary")
- Test workflow integration: how does this tool connect to what comes next?
- Scenario-based: always set up a realistic meeting/task/project context

### Money (freelancing, business use)
- Frame exercises around dollar value or client deliverables
- Prompts should produce portfolio-ready or client-ready output
- Include pricing context where relevant
- Test business judgment: "Would a client pay for this output?"

### Fun (creative, personal)
- Lower stakes, more creative freedom
- Still structured: "creative within constraints" not "do whatever"
- Emphasize surprise and delight in the AI output
- Good for introducing new tools with low pressure

### Learning (research, analysis)
- Focus on accuracy and source quality
- Test critical evaluation: "Is this AI output reliable?"
- Chain tools: research prompt into summary into study notes
- Measure comprehension, not just production

## XP Reward Scale

| Exercise Type | Base XP | Bonus Conditions |
|--------------|---------|-----------------|
| Multiple choice | 10 | +5 if first attempt correct |
| Fill-in-the-blank | 20 | +5 if answer matches "strong" tier |
| Prompt rewrite | 30 | +10 if all criteria met |
| Write from scratch | 50 | +15 if rubric score is 4/4 |
| Spaced repetition review | 15 | +10 streak bonus if reviewing on schedule |

**Daily XP cap:** None (encourage completion, not grinding)
**Streak multiplier:** 1.5x XP on days 7+, 2x on days 14+, 3x on days 21+
