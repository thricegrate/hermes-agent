# Banned Patterns

Anti-patterns to flag during content analysis. Each section lists the pattern, why it's weak, and what to use instead.

---

## Filler Phrases to Kill

Phrases that add zero information. Remove them or replace with the shorter version.

| Filler phrase | Replacement | Why it's bad |
|---------------|-------------|--------------|
| "I recently came across" | (delete, start with the thing) | Throat-clearing |
| "It's important to note that" | (delete, just state the thing) | Tells reader what to think |
| "It goes without saying" | (delete entirely) | If it goes without saying, don't say it |
| "At the end of the day" | (delete or "ultimately") | Cliche, adds nothing |
| "In today's rapidly evolving landscape" | (delete entirely) | Worst offender. AI slop marker |
| "It's worth mentioning that" | (delete, just mention it) | Permission-seeking |
| "As we all know" | (delete entirely) | If we all know it, skip it |
| "The fact of the matter is" | (delete entirely) | 7 words to say nothing |
| "In order to" | "To" | 3 unnecessary words |
| "Due to the fact that" | "Because" | 5 unnecessary words |
| "At this point in time" | "Now" | 5 unnecessary words |
| "It is interesting to note" | (delete, let the reader decide what's interesting) | Author inserting opinion as transition |
| "This is crucial because" | (delete "This is crucial because," the next sentence should stand alone) | Tells reader how to feel |
| "Needless to say" | (delete entirely) | Contradicts itself |
| "For what it's worth" | (delete entirely) | Undermines your own point |
| "I think it's safe to say" | (delete, just say it) | Hedge |
| "It should be noted that" | (delete, just note it) | Passive throat-clearing |
| "When it comes to" | (delete, name the thing directly) | Windup phrase |
| "In terms of" | (delete, name the thing directly) | Filler connector |

## Weak Transitions

Academic connectors that signal "I'm organizing my essay" instead of pulling the reader forward.

| Weak transition | Better alternatives |
|----------------|-------------------|
| "Furthermore" | "And here's the thing." / "But it doesn't stop there." |
| "Moreover" | "Even better:" / "And." |
| "Additionally" | "Plus:" / "There's more." |
| "In conclusion" | (delete, the reader knows it's the end) |
| "To summarize" | (delete, don't summarize -- land the final punch) |
| "It's worth noting that" | "Look." / "Here's what most people miss." |
| "On the other hand" | "But." / "Flip that around." |
| "In other words" | "Meaning:" / "Translation:" |
| "That being said" | "But." / "Still." |
| "Having said that" | "But." |
| "With that in mind" | "So." / (delete) |
| "As a result" | "So." / (delete, let causation be implicit) |
| "In light of this" | "So." / "Given that:" |
| "It is also important to" | "Also:" / (just say the thing) |
| "Moving on to" | (delete, use a subheading instead) |
| "Let's now turn to" | (delete, use a subheading instead) |
| "Another key aspect is" | (delete, use a subheading instead) |

**Rule:** If a transition is more than 3 words, it's probably too formal. Gary Halbert's transitions: "Now." "Look." "Anyway..." "But here's the thing." All 1-5 words.

## Adverb + Weak Verb Combos

Every adverb is a confession that the verb is too weak. Replace both with one strong verb.

| Weak combo | Strong replacement |
|------------|-------------------|
| Quickly ran | Sprinted / Dashed |
| Slowly walked | Crept / Ambled |
| Significantly reduced | Slashed / Cut |
| Carefully examined | Scrutinized / Inspected |
| Completely destroyed | Obliterated / Demolished |
| Gently touched | Caressed / Brushed |
| Loudly shouted | Bellowed / Roared |
| Rapidly increased | Surged / Spiked |
| Strongly believed | Was convinced / Knew |
| Greatly improved | Transformed / Elevated |
| Heavily invested | Poured into / Bet on |
| Clearly demonstrated | Proved / Showed |
| Fully understood | Grasped / Got |
| Actively participated | Engaged / Contributed |
| Basically means | Means |
| Essentially is | Is |
| Literally just | (delete both, say the thing) |
| Really important | Critical / Essential |
| Very unique | Rare / One-of-a-kind |
| Extremely difficult | Brutal / Punishing |

**Rule of thumb:** Search the text for "-ly" words. Each one is a suspect. Most can be killed by upgrading the verb.

## Comma Splices

Two independent clauses joined by a comma instead of a period, semicolon, or conjunction.

| Comma splice | Fix |
|-------------|-----|
| "The direction is simple, investors are moving towards..." | "The direction is simple: investors are moving towards..." (colon) |
| "She launched the product, sales doubled" | "She launched the product. Sales doubled." (period -- adds punch) |
| "AI is evolving fast, companies can't keep up" | "AI is evolving fast. Companies can't keep up." |
| "He didn't just edit it, he rewrote everything" | "He didn't just edit it -- he rewrote everything." (dash for emphasis) |

**Detection trick:** If you can replace the comma with a period and both halves are complete sentences, it's a comma splice.

**Exception:** Intentional comma splices in very short parallel clauses can work for rhythm: "I came, I saw, I conquered." But this is rare and must be clearly intentional.

## Over-Attribution Patterns

Hiding behind external authority instead of owning the content. Common in rewritten/curated articles.

| Pattern | Problem | Fix |
|---------|---------|-----|
| "According to the expert" | Who? Name them or own the point | Name the person, or present the idea as your own |
| "The original poster notes" | Reddit/forum smell | Remove attribution, present the insight directly |
| "This savvy professional" | Flattering an anonymous source | Use their name or drop the reference |
| "The innovative researcher" | Adjective-stuffing a stranger | Name or cut |
| "Industry leaders suggest" | Vague appeal to authority | Which leaders? Or just state the claim |
| "Studies show" | Which studies? Link or specify | "[Name] found that [specific finding]" or cut |
| "Experts agree" | No they don't. Which experts? | Name them, or own the opinion |

**Rule:** If you can't name the person, you probably shouldn't cite them. Either own the claim or link to the source.

## AI Slop Markers

Phrases that scream "AI-generated." Immediate credibility killers.

| Pattern | Why it's bad |
|---------|-------------|
| "In today's rapidly evolving landscape" | The #1 AI cliche |
| "Let's dive in" / "Let's dive deeper" | Overused AI opener |
| "Game-changer" | Meaningless without specifics |
| "Cutting-edge" | Says nothing about what it actually does |
| "Harness the power of" | Corporate buzzword soup |
| "Unlock the potential" | Same |
| "Seamlessly integrate" | Nothing is seamless |
| "Revolutionize your workflow" | Prove it with specifics |
| "It's not just X, it's Y" | Cliche contrast structure |
| "Whether you're a beginner or expert" | Trying to be everything to everyone |
| "Take your X to the next level" | What level? Be specific |
| "Empower" (as marketing verb) | Means nothing |
| "Delve into" | AI fingerprint |
| "Tapestry" / "Landscape" / "Realm" | AI's favorite abstract nouns |
| "Robust" / "Comprehensive" / "Holistic" | Adjectives that describe everything and nothing |
| "Embark" | AI's favorite verb for starting anything |
| "Excels" | AI inflating ordinary competence |
| "Solely" | Stiff formal word; use "only" or "just" |
| "Portray" | AI synonym cycling for "show" |
| "Ongoing" | Vague temporal filler; be specific about timeline |
| "Vigilance" | AI dramatizing ordinary attention |
| "Unleash" | Corporate hype verb, same family as "harness/unlock" |
| "Synergy" | The original corporate buzzword |
| "Multi-faceted" | AI's way of saying "complicated" without specifics |
| "Spark" / "Sparked" / "Ignite" / "Ignited" | AI fire metaphors for mild interest |
| "Navigate" (abstract) | "Navigate the complexities" is slop; literal navigation is fine |
| "Transformative" | Says everything changed without saying what changed |
| "Revolutionary" | Same problem as "game-changer" |
| "Supercharge" / "Supercharged" | Hype verb with zero specificity |

## AI Slop Phrases

Full phrases and structural patterns that scream "AI wrote this." Flag every instance.

| Pattern | Why it's bad |
|---------|-------------|
| "No X. No Y. Just Z." | AI's fake-minimalist structure. Sounds punchy, reads as template |
| "If you're serious about [goal]" | Gatekeeping CTA that every AI uses. Pressures without value |
| "To your success" | AI sign-off that no real human uses in 2026 |
| "Enter: [Thing]" | AI's dramatic introduction. Nobody talks like that |
| "Stopped me in my tracks" | Melodramatic AI reaction. Real people say "wait, what?" |
| "The catch?" / "Want to know the secret?" / "The brutal truth?" / "Here's the kicker" | Infomercial transition questions. One per piece max, never stacked |
| "Silver bullet" | Cliche. Say what it actually solves |
| "Based on the information provided" | Chatbot artifact. Delete entirely |
| "Remember that" (as sentence opener) | AI lecturing the reader. Just state the thing |
| "Navigating the landscape/complexities of" | Double slop: "navigate" (abstract) + "landscape" (abstract) |
| "Delving into the intricacies" | Triple slop: "delve" + "intricacies" + pompous framing |
| "Ready to level up?" | Generic AI CTA. Be specific about what happens next |
| "Want access?" | Lazy hook question with zero specificity |
| "The best part?" | Infomercial transition. Use a real sentence instead |
