# Engagement Prediction Checklist

Source: Adrian Solar's IG acquisition playbook. Data from 100+ Reels and thousands of social posts.

Use this checklist to predict content performance BEFORE publishing. Rate each piece against the viral patterns and flop signals below.

## Viral Pattern Scoring

Score the content against these proven high-performance patterns. Each match increases predicted performance.

| Pattern | Avg Performance | Check |
|---------|----------------|-------|
| **Specific number in hook** ("I sent 847,000 DMs...") | 340k avg views | Does the hook contain a concrete number? |
| **"Everyone's wrong" framing** | 310k avg views | Does it challenge consensus/conventional wisdom? |
| **Controversy / hot take** | 280k avg views | Will this make some people disagree? (Good.) |
| **Before/after transformation** | 250k avg views | Does it show measurable change with real metrics? |
| **Personal failure to success** | ~150k avg views | Does it include vulnerability before victory? |
| **Tool/product demo** | ~100k avg views | Does it show something working in real-time? |

**Scoring:**
- 3+ viral patterns present = high confidence, publish as-is
- 2 viral patterns = solid, consider strengthening the hook
- 1 viral pattern = average, look for a way to add a second
- 0 viral patterns = weak, rewrite before publishing

## Flop Signal Detection

Check the content against these proven failure signals. Each match decreases predicted performance.

| Signal | Avg Performance | Check |
|--------|----------------|-------|
| **Generic motivational content** ("Believe in yourself...") | 4k avg views | Could any creator in any niche post this? |
| **No hook in first 3 seconds** | 3k avg views | Could someone scroll past without knowing the value? |
| **Too polished / overproduced** | 5k avg views | Does it feel corporate instead of authentic? |
| **Self-focused** ("I achieved X") | 2k avg views | Is it about you or about the viewer's problem? |
| **No specific metrics** | Below avg | Are all claims vague ("improved results") vs specific ("went from 12% to 34%")? |
| **Consensus take** | Below avg | Is this the same thing everyone else is saying? |

**Scoring:**
- 0 flop signals = clear to publish
- 1 flop signal = fix it before publishing (usually the hook)
- 2+ flop signals = major rewrite needed, don't publish

## Quick Prediction Formula

**Predicted Performance = (Viral Pattern Count x 2) - (Flop Signal Count x 3)**

| Score | Prediction | Action |
|-------|-----------|--------|
| 6+ | Potential viral hit | Publish immediately, boost with ad spend |
| 3-5 | Strong performer | Publish, monitor for 24 hours |
| 1-2 | Average | Strengthen hook or add a viral pattern |
| 0 or below | Will underperform | Do not publish. Rewrite. |

## Hook Rewrite Priorities

When content scores poorly, fix in this order (each one has the biggest impact):

1. **Add a specific number** to the hook (biggest single lever)
2. **Find the contrarian angle** (what's your take that breaks consensus?)
3. **Add before/after proof** (show real metrics)
4. **Cut the self-focus** (reframe from "I" to "you" or "here's how")
5. **Remove corporate polish** (add personality, humor, imperfection)

## Integration with Content Review Scoring

When scoring content on the "Engaging" dimension (Part 2):
- Content scoring 6+ on this prediction formula = automatic 9-10 on Engaging
- Content scoring 3-5 = likely 7-8 on Engaging
- Content scoring 0-2 = likely 4-6 on Engaging
- Content scoring negative = automatic flag for rewrite before scoring other dimensions
